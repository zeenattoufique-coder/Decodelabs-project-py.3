# Random Password Generator

## Project Description
This is a Python project that generates a random password based on the length entered by the user. The password contains uppercase letters, lowercase letters, and numbers.

## Features
- Takes password length as input
- Generates a random password
- Uses Python's `random` and `string` modules
- Easy to use

## Technologies Used
- Python 3

## How to Run
1. Download or clone the project.
2. Open the project in VS Code.
3. Run the Python file.
4. Enter the desired password length.
5. The program will generate and display a random password.

========================================
